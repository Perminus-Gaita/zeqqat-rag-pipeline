## Requirements {/*section:requirements*/}

{/* Context: Need: Kenyan phone number (Safaricom/Airtel/Telkom), valid ID or passport, age 18+, M-Pesa account. */}

- Kenyan phone number (Safaricom, Airtel, or Telkom)
- Valid ID or Passport
- Age 18+
- M-Pesa account

## Registration Steps {/*section:registration-steps*/}

{/* Context: 7 steps: visit sportpesa.com, click Register, enter phone/ID/DOB/password, verify SMS, complete KYC, deposit minimum KSh */}

**Step 1:** Visit sportpesa.com or download the app

**Step 2:** Click "Register"

**Step 3:** Enter:
- Phone number (e.g., 0712345678)
- ID number
- Date of birth
- Create password

**Step 4:** Verify via SMS code

**Step 5:** Complete KYC (upload ID photo)

**Step 6:** Make first deposit (minimum KSh 50)

**Step 7:** Start betting!
