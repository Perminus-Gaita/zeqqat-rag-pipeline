## SportPesa SMS Codes {/*section:sms-codes*/}

{/* Context: All codes send to 79079. Mega jackpot: MJP#predictions. Midweek: MWP#predictions. Balance: BAL. Withdraw: W#Amount#PIN. */}

**Send all codes to: 79079**

### Place Mega Jackpot Bet
```
MJP#1X22222XXXX11XX12
```
(17 predictions: 1=Home, X=Draw, 2=Away)

### Place Midweek Jackpot Bet
```
MWP#1X222XX11X2X1
```
(13 predictions)

### Check Balance
```
BAL
```

### Check Results
```
(Send any text to 79079)
```

### Withdraw to M-Pesa
```
W#[Amount]#[M-Pesa PIN]
```
Example: W#5000#1234
